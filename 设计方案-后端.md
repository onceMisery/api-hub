# ApiHub 项目完整落地方案

## 一、简化架构总览

基于「少中间件」原则，对原方案做如下精简：

```
┌─────────────────────────────────────────────────────────────────┐
│                    ApiHub 精简部署架构                            │
│                                                                 │
│                       ┌──────────────┐                          │
│                       │    Nginx     │                          │
│                       │  (反向代理)   │                          │
│                       └──────┬───────┘                          │
│                    ┌─────────┴─────────┐                        │
│                    │                   │                        │
│              ┌─────▼──────┐    ┌───────▼────────┐               │
│              │ 前端 (React)│    │ ApiHub Server  │               │
│              │  静态资源    │    │ (Spring Boot)  │               │
│              └────────────┘    └───────┬────────┘               │
│                                        │                        │
│                    ┌───────────────────┼────────────┐            │
│                    │                   │            │            │
│              ┌─────▼──────┐    ┌───────▼────┐  (可选)            │
│              │  MySQL 8.0 │    │   Redis 7  │                   │
│              │  (主数据库)  │    │  (缓存/会话)│                   │
│              └────────────┘    └────────────┘                   │
│                                                                 │
│  去掉的中间件: Elasticsearch → MySQL全文索引                      │
│              MinIO → 本地文件存储                                 │
│              RabbitMQ → Spring @Async                            │
│              Redis可选 → 无Redis时用Caffeine本地缓存               │
└─────────────────────────────────────────────────────────────────┘
```

## 二、技术栈选型

| 层次        | 技术                      | 版本     | 说明                       |
|-----------|-------------------------|--------|--------------------------|
| **后端框架**  | Spring Boot             | 3.3.x  | Java 21, Virtual Threads |
| **ORM**   | MyBatis-Plus            | 3.5.x  | 简洁高效                     |
| **数据库**   | MySQL                   | 8.0+   | JSON列 + 全文索引             |
| **缓存**    | Caffeine / Redis        | -      | 无Redis时降级为本地缓存           |
| **认证**    | JWT + Spring Security   | -      | 无状态Token认证               |
| **API文档** | 自身吃自身狗粮                 | -      | 自动生成                     |
| **前端框架**  | React 18                | 18.3.x | TypeScript 5             |
| **构建工具**  | Vite                    | 5.x    | 极速开发体验                   |
| **UI组件库** | Ant Design 5            | 5.x    | + Tailwind CSS           |
| **代码编辑器** | Monaco Editor           | -      | VS Code同款                |
| **状态管理**  | Zustand                 | 4.x    | 轻量级                      |
| **请求库**   | @tanstack/react-query   | 5.x    | 服务端状态管理                  |
| **部署**    | Docker + Docker Compose | -      | 一键启动                     |

## 四、数据库设计 (MySQL 完整DDL)

```sql
-- ================================================================
-- ApiHub 数据库初始化脚本 (MySQL 8.0+)
-- 文件: db/migration/V1__init_schema.sql
-- ================================================================

-- -----------------------------------------
-- 1. 用户体系
-- -----------------------------------------

CREATE TABLE `sys_user` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
    `username`        VARCHAR(64)  NOT NULL COMMENT '用户名',
    `password`        VARCHAR(256) NOT NULL COMMENT '密码(BCrypt)',
    `nickname`        VARCHAR(64)  NOT NULL COMMENT '昵称',
    `email`           VARCHAR(128) DEFAULT NULL COMMENT '邮箱',
    `avatar_url`      VARCHAR(512) DEFAULT NULL COMMENT '头像URL',
    `status`          TINYINT      NOT NULL DEFAULT 1 COMMENT '1-正常 0-禁用',
    `last_login_at`   DATETIME     DEFAULT NULL COMMENT '最后登录时间',
    `created_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    `updated_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    `deleted`         TINYINT      NOT NULL DEFAULT 0,
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_username` (`username`),
    UNIQUE KEY `uk_email` (`email`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='用户表';

-- -----------------------------------------
-- 2. 空间/团队
-- -----------------------------------------

CREATE TABLE `space` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
    `name`            VARCHAR(128) NOT NULL COMMENT '空间名称',
    `description`     TEXT         DEFAULT NULL COMMENT '描述',
    `avatar_url`      VARCHAR(512) DEFAULT NULL COMMENT '头像',
    `visibility`      TINYINT      NOT NULL DEFAULT 0 COMMENT '0-私有 1-公开',
    `creator_id`      BIGINT       NOT NULL COMMENT '创建人',
    `created_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    `updated_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    `deleted`         TINYINT      NOT NULL DEFAULT 0,
    PRIMARY KEY (`id`),
    KEY `idx_creator` (`creator_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='空间/团队';

CREATE TABLE `space_member` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
    `space_id`        BIGINT       NOT NULL,
    `user_id`         BIGINT       NOT NULL,
    `role`            VARCHAR(32)  NOT NULL COMMENT 'admin/developer/visitor',
    `created_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_space_user` (`space_id`, `user_id`),
    KEY `idx_user` (`user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='空间成员';

-- -----------------------------------------
-- 3. 项目
-- -----------------------------------------

CREATE TABLE `project` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
    `space_id`        BIGINT       NOT NULL,
    `name`            VARCHAR(128) NOT NULL COMMENT '项目名称',
    `description`     TEXT         DEFAULT NULL,
    `visibility`      TINYINT      NOT NULL DEFAULT 0 COMMENT '0-私有 1-空间可见 2-公开',
    `project_type`    VARCHAR(32)  NOT NULL DEFAULT 'http' COMMENT 'http/dubbo/grpc',
    `creator_id`      BIGINT       NOT NULL,
    `created_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    `updated_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    `deleted`         TINYINT      NOT NULL DEFAULT 0,
    PRIMARY KEY (`id`),
    KEY `idx_space` (`space_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='项目';

CREATE TABLE `project_member` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
    `project_id`      BIGINT       NOT NULL,
    `user_id`         BIGINT       NOT NULL,
    `role`            VARCHAR(32)  NOT NULL COMMENT 'admin/developer/visitor',
    `created_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_project_user` (`project_id`, `user_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='项目成员';

-- -----------------------------------------
-- 4. 模块
-- -----------------------------------------

CREATE TABLE `module` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
    `project_id`      BIGINT       NOT NULL,
    `name`            VARCHAR(128) NOT NULL COMMENT '模块名称',
    `description`     TEXT         DEFAULT NULL,
    `sort_order`      INT          NOT NULL DEFAULT 0,
    `push_token`      VARCHAR(128) DEFAULT NULL COMMENT 'DocForge推送令牌',
    `creator_id`      BIGINT       NOT NULL,
    `created_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    `updated_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    `deleted`         TINYINT      NOT NULL DEFAULT 0,
    PRIMARY KEY (`id`),
    UNIQUE KEY `uk_push_token` (`push_token`),
    KEY `idx_project` (`project_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='模块';

-- -----------------------------------------
-- 5. 接口分组 (Controller/Service)
-- -----------------------------------------

CREATE TABLE `api_group` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
    `module_id`       BIGINT       NOT NULL,
    `parent_id`       BIGINT       NOT NULL DEFAULT 0 COMMENT '父分组ID, 0=顶级',
    `name`            VARCHAR(256) NOT NULL COMMENT '分组名称',
    `description`     TEXT         DEFAULT NULL,
    `author`          VARCHAR(64)  DEFAULT NULL,
    `sort_order`      INT          NOT NULL DEFAULT 0,
    `created_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    `updated_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    `deleted`         TINYINT      NOT NULL DEFAULT 0,
    PRIMARY KEY (`id`),
    KEY `idx_module` (`module_id`),
    KEY `idx_parent` (`parent_id`)
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='接口分组';

-- -----------------------------------------
-- 6. 接口端点 (核心表)
-- -----------------------------------------

CREATE TABLE `api_endpoint` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
    `group_id`        BIGINT       NOT NULL COMMENT '所属分组',
    `module_id`       BIGINT       NOT NULL COMMENT '所属模块',
    `name`            VARCHAR(256) NOT NULL COMMENT '接口名称',
    `description`     TEXT         DEFAULT NULL COMMENT '接口描述',
    `http_method`     VARCHAR(16)  NOT NULL COMMENT 'GET/POST/PUT/DELETE/PATCH',
    `url`             VARCHAR(512) NOT NULL COMMENT '接口路径',
    `content_type`    VARCHAR(128) DEFAULT 'application/json',
    `author`          VARCHAR(64)  DEFAULT NULL COMMENT '负责人',
    `status`          VARCHAR(32)  NOT NULL DEFAULT 'developing' COMMENT 'developing/testing/released/deprecated',
    `deprecated`      TINYINT      NOT NULL DEFAULT 0 COMMENT '是否废弃',
    `version`         VARCHAR(32)  DEFAULT NULL COMMENT '接口版本号',
    `sort_order`      INT          NOT NULL DEFAULT 0,
    `is_locked`       TINYINT      NOT NULL DEFAULT 0 COMMENT '编辑锁',
    `locked_by`       BIGINT       DEFAULT NULL,
    `locked_at`       DATETIME     DEFAULT NULL,
    `created_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP,
    `updated_at`      DATETIME     NOT NULL DEFAULT CURRENT_TIMESTAMP ON UPDATE CURRENT_TIMESTAMP,
    `deleted`         TINYINT      NOT NULL DEFAULT 0,
    PRIMARY KEY (`id`),
    KEY `idx_group` (`group_id`),
    KEY `idx_module` (`module_id`),
    KEY `idx_method_url` (`http_method`, `url`),
    FULLTEXT KEY `ft_name_desc` (`name`, `description`) WITH PARSER ngram  -- 全文搜索
) ENGINE=InnoDB DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_unicode_ci COMMENT='接口端点';

-- -----------------------------------------
-- 7. 接口请求参数
-- -----------------------------------------

CREATE TABLE `api_parameter` (
    `id`              BIGINT       NOT NULL AUTO_INCREMENT,
```

```java
    public R<?> handleException(Exception e) {
        log.error("系统异常", e);
```

```java
package com.apihub.security;

import io.jsonwebtoken.*;
import io.jsonwebtoken.security.Keys;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Component;

import javax.crypto.SecretKey;
import java.nio.charset.StandardCharsets;
import java.util.Date;

@Component
public class JwtUtil {

    @Value("${apihub.jwt.secret}")
    private String secret;

    @Value("${apihub.jwt.expiration}")
    private long expiration;

    @Value("${apihub.jwt.refresh-expiration}")
    private long refreshExpiration;

    private SecretKey getKey() {
        return Keys.hmacShaKeyFor(secret.getBytes(StandardCharsets.UTF_8));
    }

    public String generateToken(Long userId, String username) {
        return Jwts.builder()
                .subject(String.valueOf(userId))
                .claim("username", username)
                .issuedAt(new Date())
                .expiration(new Date(System.currentTimeMillis() + expiration))
                .signWith(getKey())
                .compact();
    }

    public String generateRefreshToken(Long userId) {
        return Jwts.builder()
                .subject(String.valueOf(userId))
                .claim("type", "refresh")
                .issuedAt(new Date())
                .expiration(new Date(System.currentTimeMillis() + refreshExpiration))
                .signWith(getKey())
                .compact();
    }

    public Claims parseToken(String token) {
        return Jwts.parser()
                .verifyWith(getKey())
                .build()
                .parseSignedClaims(token)
                .getPayload();
    }

    public boolean validateToken(String token) {
        try {
            parseToken(token);
            return true;
        } catch (JwtException e) {
            return false;
        }
    }

    public Long getUserId(String token) {
        return Long.parseLong(parseToken(token).getSubject());
    }

    public String getUsername(String token) {
        return parseToken(token).get("username", String.class);
    }
}
```

```java
package com.apihub.security;

import com.apihub.common.context.UserContext;
import jakarta.servlet.FilterChain;
import jakarta.servlet.ServletException;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import lombok.RequiredArgsConstructor;
import org.springframework.security.authentication.UsernamePasswordAuthenticationToken;
import org.springframework.security.core.context.SecurityContextHolder;
import org.springframework.stereotype.Component;
import org.springframework.util.StringUtils;
import org.springframework.web.filter.OncePerRequestFilter;

import java.io.IOException;
import java.util.Collections;

@Component
@RequiredArgsConstructor
public class JwtAuthenticationFilter extends OncePerRequestFilter {

    private final JwtUtil jwtUtil;

    @Override
    protected void doFilterInternal(HttpServletRequest request,
                                    HttpServletResponse response,
                                    FilterChain filterChain) throws ServletException, IOException {
        try {
            String token = extractToken(request);
            if (StringUtils.hasText(token) && jwtUtil.validateToken(token)) {
                Long userId = jwtUtil.getUserId(token);
                String username = jwtUtil.getUsername(token);

                UserContext.set(userId, username);

                UsernamePasswordAuthenticationToken authentication =
                        new UsernamePasswordAuthenticationToken(userId, null, Collections.emptyList());
                SecurityContextHolder.getContext().setAuthentication(authentication);
            }
            filterChain.doFilter(request, response);
        } finally {
            UserContext.clear();
        }
    }

    private String extractToken(HttpServletRequest request) {
        String bearerToken = request.getHeader("Authorization");
        if (StringUtils.hasText(bearerToken) && bearerToken.startsWith("Bearer ")) {
            return bearerToken.substring(7);
        }
        return null;
    }
}
```

```java
import org.springframework.security.config.http.SessionCreationPolicy;
import org.springframework.security.crypto.bcrypt.BCryptPasswordEncoder;
import org.springframework.security.crypto.password.PasswordEncoder;
import org.springframework.security.web.SecurityFilterChain;
import org.springframework.security.web.authentication.UsernamePasswordAuthenticationFilter;
import org.springframework.web.cors.CorsConfiguration;
import org.springframework.web.cors.CorsConfigurationSource;
import org.springframework.web.cors.UrlBasedCorsConfigurationSource;

import java.util.List;

@Configuration
@EnableWebSecurity
@EnableMethodSecurity
@RequiredArgsConstructor
public class SecurityConfig {

    private final JwtAuthenticationFilter jwtFilter;

    @Bean
    public SecurityFilterChain securityFilterChain(HttpSecurity http) throws Exception {
        http
                .csrf(AbstractHttpConfigurer::disable)
                .cors(cors -> cors.configurationSource(corsConfigurationSource()))
                .sessionManagement(session -> session.sessionCreationPolicy(SessionCreationPolicy.STATELESS))
                .authorizeHttpRequests(auth -> auth
                        // 公开接口
                        .requestMatchers("/api/auth/**").permitAll()
                        .requestMatchers("/api/share/**").permitAll()
                        .requestMatchers("/mock/**").permitAll()
                        .requestMatchers("/api/push/**").permitAll()
                        .requestMatchers(HttpMethod.OPTIONS, "/**").permitAll()
                        // 其余需要认证
                        .anyRequest().authenticated()
                )
                .addFilterBefore(jwtFilter, UsernamePasswordAuthenticationFilter.class);

        return http.build();
    }

    @Bean
    public PasswordEncoder passwordEncoder() {
        return new BCryptPasswordEncoder();
    }

    @Bean
    public CorsConfigurationSource corsConfigurationSource() {
        CorsConfiguration config = new CorsConfiguration();
        config.setAllowedOriginPatterns(List.of("*"));
        config.setAllowedMethods(List.of("GET", "POST", "PUT", "DELETE", "PATCH", "OPTIONS"));
        config.setAllowedHeaders(List.of("*"));
        config.setAllowCredentials(true);
        config.setMaxAge(3600L);

        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/**", config);
        return source;
    }
}
```

```java
package com.apihub.service;

import com.apihub.common.exception.BizException;
import com.apihub.mapper.SysUserMapper;
import com.apihub.model.dto.auth.*;
import com.apihub.model.entity.SysUser;
import com.apihub.model.vo.LoginVO;
import com.apihub.security.JwtUtil;
import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import lombok.RequiredArgsConstructor;
import org.springframework.security.crypto.password.PasswordEncoder;
import org.springframework.stereotype.Service;

import java.time.LocalDateTime;

@Service
@RequiredArgsConstructor
public class AuthService {

    private final SysUserMapper userMapper;
    private final PasswordEncoder passwordEncoder;
    private final JwtUtil jwtUtil;

    public LoginVO login(LoginDTO dto) {
        SysUser user = userMapper.selectOne(
                new LambdaQueryWrapper<SysUser>().eq(SysUser::getUsername, dto.getUsername()));
        if (user == null || !passwordEncoder.matches(dto.getPassword(), user.getPassword())) {
            throw new BizException("用户名或密码错误");
        }
        if (user.getStatus() != 1) {
            throw new BizException("账号已被禁用");
        }

        // 更新登录时间
        user.setLastLoginAt(LocalDateTime.now());
        userMapper.updateById(user);

        String token = jwtUtil.generateToken(user.getId(), user.getUsername());
        String refreshToken = jwtUtil.generateRefreshToken(user.getId());

        LoginVO vo = new LoginVO();
        vo.setToken(token);
        vo.setRefreshToken(refreshToken);
        vo.setUserId(user.getId());
        vo.setUsername(user.getUsername());
        vo.setNickname(user.getNickname());
        vo.setAvatarUrl(user.getAvatarUrl());
        return vo;
    }

    public void register(RegisterDTO dto) {
        // 检查用户名唯一
        Long count = userMapper.selectCount(
                new LambdaQueryWrapper<SysUser>().eq(SysUser::getUsername, dto.getUsername()));
        if (count > 0) {
            throw new BizException("用户名已存在");
        }

        SysUser user = new SysUser();
        user.setUsername(dto.getUsername());
        user.setPassword(passwordEncoder.encode(dto.getPassword()));
        user.setNickname(dto.getNickname());
        user.setEmail(dto.getEmail());
        user.setStatus(1);
        userMapper.insert(user);
    }
}
```

```java
package com.apihub.service;

import com.apihub.common.exception.BizException;
import com.apihub.mapper.*;
import com.apihub.model.entity.*;
import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import lombok.RequiredArgsConstructor;
import org.springframework.cache.annotation.Cacheable;
import org.springframework.stereotype.Service;

@Service
@RequiredArgsConstructor
public class PermissionService {

    private final SpaceMemberMapper spaceMemberMapper;
    private final ProjectMemberMapper projectMemberMapper;
    private final ProjectMapper projectMapper;
    private final ModuleMapper moduleMapper;
    private final ApiGroupMapper apiGroupMapper;

    /**
     * 检查用户在指定资源上的角色 (向上继承)
     */
    @Cacheable(value = "permission", key = "#userId + ':' + #resourceType + ':' + #resourceId")
    public String getEffectiveRole(Long userId, String resourceType, Long resourceId) {
        return switch (resourceType) {
            case "space" -> getSpaceRole(userId, resourceId);
            case "project" -> getProjectRole(userId, resourceId);
            case "module" -> {
                Module module = moduleMapper.selectById(resourceId);
                if (module == null) yield null;
                yield getProjectRole(userId, module.getProjectId());
            }
            case "endpoint" -> {
                // endpoint -> group -> module -> project -> space
                ApiGroup group = apiGroupMapper.selectById(resourceId);
                if (group == null) yield null;
                Module module = moduleMapper.selectById(group.getModuleId());
                if (module == null) yield null;
                yield getProjectRole(userId, module.getProjectId());
            }
            default -> null;
        };
    }

    private String getSpaceRole(Long userId, Long spaceId) {
        SpaceMember member = spaceMemberMapper.selectOne(
                new LambdaQueryWrapper<SpaceMember>()
                        .eq(SpaceMember::getSpaceId, spaceId)
                        .eq(SpaceMember::getUserId, userId));
        return member != null ? member.getRole() : null;
    }

    private String getProjectRole(Long userId, Long projectId) {
        // 先查项目级别
        ProjectMember pm = projectMemberMapper.selectOne(
                new LambdaQueryWrapper<ProjectMember>()
                        .eq(ProjectMember::getProjectId, projectId)
                        .eq(ProjectMember::getUserId, userId));
        if (pm != null) return pm.getRole();

        // 继承空间权限
        Project project = projectMapper.selectById(projectId);
        if (project != null) {
            return getSpaceRole(userId, project.getSpaceId());
        }
        return null;
    }

    /**
     * 断言用户有权限, 否则抛异常
     */
    public void checkPermission(Long userId, String resourceType, Long resourceId, String... allowedRoles) {
        String role = getEffectiveRole(userId, resourceType, resourceId);
        if (role == null) {
            throw new BizException(403, "无权限访问此资源");
        }
        boolean allowed = false;
        for (String r : allowedRoles) {
            if (r.equals(role)) {
                allowed = true;
                break;
            }
        }
        if (!allowed) {
            throw new BizException(403, "权限不足");
        }
    }

    public void checkCanView(Long userId, String resourceType, Long resourceId) {
        checkPermission(userId, resourceType, resourceId, "admin", "developer", "visitor");
    }

    public void checkCanEdit(Long userId, String resourceType, Long resourceId) {
        checkPermission(userId, resourceType, resourceId, "admin", "developer");
    }

    public void checkIsAdmin(Long userId, String resourceType, Long resourceId) {
        checkPermission(userId, resourceType, resourceId, "admin");
    }
}
```

```java
// ============================================================
// com/apihub/service/EndpointService.java - 接口管理服务
// ============================================================
package com.apihub.service;

import com.apihub.common.context.UserContext;
import com.apihub.common.exception.BizException;
import com.apihub.mapper.*;
import com.apihub.model.dto.endpoint.*;
import com.apihub.model.entity.*;
import com.apihub.model.vo.*;
import com.baomidou.mybatisplus.core.conditions.query.LambdaQueryWrapper;
import com.fasterxml.jackson.databind.ObjectMapper;
import lombok.RequiredArgsConstructor;
import org.springframework.stereotype.Service;
import org.springframework.transaction.annotation.Transactional;

import java.util.*;
import java.util.stream.Collectors;

@Service
@RequiredArgsConstructor
public class EndpointService {

    private final ApiEndpointMapper endpointMapper;
    private final ApiParameterMapper parameterMapper;
    private final ApiResponseMapper responseMapper;
    private final ApiVersionMapper versionMapper;
    private final PermissionService permissionService;
    private final ObjectMapper objectMapper;

    /**
     * 获取接口详情 (含树形参数和响应)
     */
    public EndpointDetailVO getDetail(Long endpointId) {
        ApiEndpoint endpoint = endpointMapper.selectById(endpointId);
        if (endpoint == null) throw new BizException("接口不存在");

        EndpointDetailVO vo = new EndpointDetailVO();
        // 复制基础属性
        vo.setId(endpoint.getId());
        vo.setGroupId(endpoint.getGroupId());
        vo.setModuleId(endpoint.getModuleId());
        vo.setName(endpoint.getName());
        vo.setDescription(endpoint.getDescription());
        vo.setHttpMethod(endpoint.getHttpMethod());
        vo.setUrl(endpoint.getUrl());
        vo.setContentType(endpoint.getContentType());
        vo.setAuthor(endpoint.getAuthor());
        vo.setStatus(endpoint.getStatus());
        vo.setVersion(endpoint.getVersion());
        vo.setDeprecated(endpoint.getDeprecated() == 1);
        vo.setCreatedAt(endpoint.getCreatedAt());
        vo.setUpdatedAt(endpoint.getUpdatedAt());

        // 构建参数树
        List<ApiParameter> allParams = parameterMapper.selectList(
                new LambdaQueryWrapper<ApiParameter>()
                        .eq(ApiParameter::getEndpointId, endpointId)
                        .orderByAsc(ApiParameter::getSortOrder));

        Map<String, List<ParamTreeVO>> grouped = buildParamTree(allParams);
        vo.setHeaderParams(grouped.getOrDefault("header", List.of()));
        vo.setPathParams(grouped.getOrDefault("path", List.of()));
        vo.setQueryParams(grouped.getOrDefault("query", List.of()));
        vo.setBodyParams(grouped.getOrDefault("body", List.of()));

        // 构建响应树
        List<ApiResponse> allResponses = responseMapper.selectList(
                new LambdaQueryWrapper<ApiResponse>()
                        .eq(ApiResponse::getEndpointId, endpointId)
                        .orderByAsc(ApiResponse::getSortOrder));
        vo.setResponses(buildResponseTree(allResponses));

        return vo;
    }

    /**
     * 保存接口 (新增/更新 + 参数 + 响应, 事务性保存)
     */
    @Transactional
    public Long saveEndpoint(EndpointSaveDTO dto) {
        Long userId = UserContext.getUserId();
        permissionService.checkCanEdit(userId, "module", dto.getModuleId());

        ApiEndpoint endpoint;
        boolean isNew = (dto.getId() == null);

        if (isNew) {
            endpoint = new ApiEndpoint();
        } else {
            endpoint = endpointMapper.selectById(dto.getId());
            if (endpoint == null) throw new BizException("接口不存在");
            // 检查编辑锁
            if (endpoint.getIsLocked() == 1 && !endpoint.getLockedBy().equals(userId)) {
                throw new BizException("接口已被其他人锁定编辑");
            }
        }

        // 设置基础属性
        endpoint.setGroupId(dto.getGroupId());
        endpoint.setModuleId(dto.getModuleId());
        endpoint.setName(dto.getName());
        endpoint.setDescription(dto.getDescription());
        endpoint.setHttpMethod(dto.getHttpMethod().toUpperCase());
        endpoint.setUrl(dto.getUrl());
        endpoint.setContentType(dto.getContentType());
        endpoint.setAuthor(dto.getAuthor());
        endpoint.setStatus(dto.getStatus() != null ? dto.getStatus() : "developing");

        if (isNew) {
            endpointMapper.insert(endpoint);
        } else {
            endpointMapper.updateById(endpoint);
            // 保存版本快照 (更新时)
            saveVersionSnapshot(endpoint);
        }

        Long endpointId = endpoint.getId();

        // 保存参数 (先删后插)
        parameterMapper.delete(new LambdaQueryWrapper<ApiParameter>()
                .eq(ApiParameter::getEndpointId, endpointId));
        if (dto.getParameters() != null) {
            saveParamsRecursively(dto.getParameters(), endpointId, 0L);
        }

        // 保存响应 (先删后插)
        responseMapper.delete(new LambdaQueryWrapper<ApiResponse>()
                .eq(ApiResponse::getEndpointId, endpointId));
        if (dto.getResponses() != null) {
            saveResponsesRecursively(dto.getResponses(), endpointId, 0L);
        }

        return endpointId;
    }

    /**
     * 递归保存参数 (支持嵌套)
     */
    private void saveParamsRecursively(List<ParamDTO> params, Long endpointId, Long parentId) {
        if (params == null) return;
        for (int i = 0; i < params.size(); i++) {
            ParamDTO dto = params.get(i);
            ApiParameter param = new ApiParameter();
            param.setEndpointId(endpointId);
            param.setParentId(parentId);
            param.setParamIn(dto.getParamIn());
            param.setName(dto.getName());
            param.setType(dto.getType());
            param.setRequired(dto.getRequired() != null ? dto.getRequired() : 0);
            param.setMaxLength(dto.getMaxLength());
            param.setDescription(dto.getDescription());
            param.setExample(dto.getExample());
            param.setDefaultValue(dto.getDefaultValue());
            param.setEnumId(dto.getEnumId());
            param.setSortOrder(i);
            parameterMapper.insert(param);

            // 递归子参数
            if (dto.getChildren() != null && !dto.getChildren().isEmpty()) {
                saveParamsRecursively(dto.getChildren(), endpointId, param.getId());
            }
        }
    }

    private void saveResponsesRecursively(List<ResponseFieldDTO> fields, Long endpointId, Long parentId) {
        if (fields == null) return;
        for (int i = 0; i < fields.size(); i++) {
            ResponseFieldDTO dto = fields.get(i);
            ApiResponse resp = new ApiResponse();
            resp.setEndpointId(endpointId);
            resp.setParentId(parentId);
            resp.setHttpCode(dto.getHttpCode() != null ? dto.getHttpCode() : 200);
            resp.setName(dto.getName());
            resp.setType(dto.getType());
            resp.setDescription(dto.getDescription());
            resp.setExample(dto.getExample());
            resp.setSortOrder(i);
            responseMapper.insert(resp);

            if (dto.getChildren() != null && !dto.getChildren().isEmpty()) {
                saveResponsesRecursively(dto.getChildren(), endpointId, resp.getId());
            }
        }
    }

    /**
     * 保存版本快照
     */
    private void saveVersionSnapshot(ApiEndpoint endpoint) {
        try {
            EndpointDetailVO detail = getDetail(endpoint.getId());
            String snapshotJson = objectMapper.writeValueAsString(detail);

            // 自动生成版本号
            Long versionCount = versionMapper.selectCount(
                    new LambdaQueryWrapper<ApiVersion>()
                            .eq(ApiVersion::getEndpointId, endpoint.getId()));
            String versionLabel = "v" + (versionCount + 1);

            ApiVersion version = new ApiVersion();
            version.setEndpointId(endpoint.getId());
            version.setVersionLabel(versionLabel);
            version.setSnapshotData(snapshotJson);
            version.setOperatorId(UserContext.getUserId());
            versionMapper.insert(version);
        } catch (Exception e) {
            // 版本保存失败不影响主流程
        }
    }

    /**
     * 构建参数树
     */
    private Map<String, List<ParamTreeVO>> buildParamTree(List<ApiParameter> allParams) {
        Map<Long, List<ApiParameter>> childMap = allParams.stream()
                .collect(Collectors.groupingBy(ApiParameter::getParentId));

        List<ApiParameter> roots = childMap.getOrDefault(0L, List.of());

        return roots.stream()
                .map(p -> toParamTreeVO(p, childMap))
                .collect(Collectors.groupingBy(ParamTreeVO::getParamIn));
    }

    private ParamTreeVO toParamTreeVO(ApiParameter param, Map<Long, List<ApiParameter>> childMap) {
        ParamTreeVO vo = new ParamTreeVO();
        vo.setId(param.getId());

        vo.setName(param.getName());

        vo.setMaxLength(param.getMaxLength());
        vo.setDescription(param.getDescription());
        vo.setExample(param.getExample());
        vo.setDefaultValue(param.getDefaultValue());

        List<ApiParameter> children = childMap.getOrDefault(param.getId(), List.of());
        vo.setChildren(children.stream()
                .map(c -> toParamTreeVO(c, childMap))
                .collect(Collectors.toList()));

                .map(r -> toResponseTreeVO(r, childMap))
                .collect(Collectors.toList());
```

```java
package com.apihub.debug;

import com.apihub.common.context.UserContext;
import com.apihub.mapper.DebugHistoryMapper;

import lombok.RequiredArgsConstructor;

import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.time.Duration;
import java.util.LinkedHashMap;
import java.util.Map;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

@Slf4j
@Service
@RequiredArgsConstructor
public class DebugProxyService {

    private final EnvironmentMapper environmentMapper;
    private final DebugHistoryMapper debugHistoryMapper;
    private final ObjectMapper objectMapper;

    private static final Pattern VARIABLE_PATTERN = Pattern.compile("\\{\\{(\\w+)}}");

    private final HttpClient httpClient = HttpClient.newBuilder()
            .connectTimeout(Duration.ofSeconds(10))
            .followRedirects(HttpClient.Redirect.NORMAL)
            .build();

    public DebugResponseVO send(DebugRequestDTO request) throws Exception {
        // 1. 加载环境配置
        Environment env = null;
        Map<String, String> variables = new LinkedHashMap<>();
        if (request.getEnvId() != null) {
            env = environmentMapper.selectById(request.getEnvId());
            if (env != null && env.getVariables() != null) {
                variables.putAll(objectMapper.readValue(
                        objectMapper.writeValueAsString(env.getVariables()), Map.class));
            }
        }

        // 2. 构建完整URL
        String baseUrl = (env != null) ? env.getBaseUrl() : "";
        String fullUrl = resolveVariables(baseUrl + request.getUrl(), variables);

        // 添加查询参数
        if (request.getQueryParams() != null && !request.getQueryParams().isEmpty()) {
            StringBuilder sb = new StringBuilder(fullUrl);
            sb.append(fullUrl.contains("?") ? "&" : "?");
            request.getQueryParams().forEach((k, v) ->
                    sb.append(k).append("=").append(resolveVariables(v, variables)).append("&"));
            fullUrl = sb.substring(0, sb.length() - 1);
        }

        // 3. 合并请求头
        Map<String, String> mergedHeaders = new LinkedHashMap<>();
        if (env != null && env.getGlobalHeaders() != null) {
            mergedHeaders.putAll(objectMapper.readValue(
                    objectMapper.writeValueAsString(env.getGlobalHeaders()), Map.class));
        }
        if (request.getHeaders() != null) {
            mergedHeaders.putAll(request.getHeaders());
        }
        // 变量替换
        mergedHeaders.replaceAll((k, v) -> resolveVariables(v, variables));

        // 4. 请求体变量替换
        String body = request.getBody();
        if (body != null) {
            body = resolveVariables(body, variables);
        }

        // 5. 构建HTTP请求
        HttpRequest.Builder httpReqBuilder = HttpRequest.newBuilder()
                .uri(URI.create(fullUrl))
                .timeout(Duration.ofSeconds(30));

        mergedHeaders.forEach(httpReqBuilder::header);

        String method = request.getMethod().toUpperCase();
        HttpRequest.BodyPublisher bodyPublisher = (body != null && !body.isBlank())
                ? HttpRequest.BodyPublishers.ofString(body)
                : HttpRequest.BodyPublishers.noBody();

        httpReqBuilder.method(method, bodyPublisher);

        // 6. 发送请求
        long startTime = System.nanoTime();
        HttpResponse<String> response = httpClient.send(httpReqBuilder.build(),
                HttpResponse.BodyHandlers.ofString());
        long durationMs = (System.nanoTime() - startTime) / 1_000_000;

        // 7. 构建响应
        DebugResponseVO vo = new DebugResponseVO();
        vo.setStatusCode(response.statusCode());
        vo.setHeaders(response.headers().map());
        vo.setBody(response.body());
        vo.setDurationMs(durationMs);
        vo.setBodySize(response.body() != null ? response.body().getBytes().length : 0);

        // 8. 异步保存调试历史
        saveHistoryAsync(request, vo);

        return vo;
    }

    /**
     * 变量替换: {{varName}} → 实际值
     */
    private String resolveVariables(String template, Map<String, String> variables) {
        if (template == null || variables.isEmpty()) return template;
        Matcher matcher = VARIABLE_PATTERN.matcher(template);
        StringBuilder sb = new StringBuilder();
        while (matcher.find()) {
            String varName = matcher.group(1);
            String value = variables.getOrDefault(varName, matcher.group(0));
            matcher.appendReplacement(sb, Matcher.quoteReplacement(value));
        }
        matcher.appendTail(sb);
        return sb.toString();
    }

    @Async
    public void saveHistoryAsync(DebugRequestDTO request, DebugResponseVO response) {
        try {
            DebugHistory history = new DebugHistory();
            history.setProjectId(request.getProjectId());
            history.setEndpointId(request.getEndpointId());
            history.setUserId(UserContext.getUserId());
            history.setMethod(request.getMethod());
            history.setUrl(request.getUrl());
            history.setRequestHeaders(objectMapper.writeValueAsString(request.getHeaders()));
            history.setRequestBody(request.getBody());
            history.setResponseCode(response.getStatusCode());
            history.setResponseBody(response.getBody());
            history.setDurationMs(response.getDurationMs());
            debugHistoryMapper.insert(history);
        } catch (Exception e) {
            log.warn("保存调试历史失败", e);
        }
    }
```

```java
public class MockEngine {

    private final ModuleMapper moduleMapper;
    private final ApiEndpointMapper endpointMapper;
    private final MockRuleMapper mockRuleMapper;
    private final ApiResponseMapper responseMapper;
    private final ObjectMapper objectMapper;

    private final AntPathMatcher pathMatcher = new AntPathMatcher();

    public MockResult handle(String projectToken, String method, String path,
                             Map<String, String> headers, Map<String, String> queryParams, String body) {

        // 1. 根据token找模块
        Module module = moduleMapper.selectOne(
                new LambdaQueryWrapper<Module>()
                        .eq(Module::getPushToken, projectToken));
        if (module == null) {
            return MockResult.notFound("模块不存在");
        }

        // 2. 匹配接口 (支持路径参数: /api/users/{id})
        List<ApiEndpoint> endpoints = endpointMapper.findByModuleAndMethod(module.getId(), method.toUpperCase());
        ApiEndpoint matchedEndpoint = null;
        for (ApiEndpoint ep : endpoints) {
            if (pathMatcher.match(ep.getUrl(), path) || ep.getUrl().equals(path)) {
                matchedEndpoint = ep;
                break;
            }
        }
        if (matchedEndpoint == null) {
            return MockResult.notFound("接口不存在: " + method + " " + path);
        }

        // 3. 查找Mock规则
        List<MockRule> rules = mockRuleMapper.selectList(
                new LambdaQueryWrapper<MockRule>()
                        .eq(MockRule::getEndpointId, matchedEndpoint.getId())
                        .eq(MockRule::getIsActive, 1)
                        .orderByAsc(MockRule::getSortOrder));

        // 4. 条件匹配
        MockRule matchedRule = matchRule(rules, headers, queryParams, body);

        if (matchedRule != null) {
            // 使用匹配到的规则
            return MockResult.builder()
                    .statusCode(matchedRule.getResponseCode())
                    .headers(matchedRule.getResponseHeaders() != null ?
                            matchedRule.getResponseHeaders() : Map.of("Content-Type", "application/json"))
                    .body(matchedRule.getResponseBody())
                    .delayMs(matchedRule.getDelayMs())
                    .build();
        }

        // 5. 无规则时自动生成Mock数据
        String autoMockBody = generateAutoMock(matchedEndpoint.getId());
        return MockResult.builder()
                .statusCode(200)
                .headers(Map.of("Content-Type", "application/json"))
                .body(autoMockBody)
                .delayMs(0)
                .build();
    }

    /**
     * 条件匹配 Mock 规则
     */
    private MockRule matchRule(List<MockRule> rules, Map<String, String> headers,
                               Map<String, String> queryParams, String body) {
        for (MockRule rule : rules) {
            if (rule.getMatchCondition() == null || rule.getMatchCondition().isEmpty()) {
                return rule; // 无条件规则，直接匹配
            }

            boolean allMatch = true;
            for (Map.Entry<String, Object> entry : rule.getMatchCondition().entrySet()) {
                String key = entry.getKey();
                String expected = String.valueOf(entry.getValue());

                String actual = null;
                if (key.startsWith("header.")) {
                    actual = headers.get(key.substring(7));
                } else if (key.startsWith("query.")) {
                    actual = queryParams.get(key.substring(6));
                }

                if (!expected.equals(actual)) {
                    allMatch = false;
                    break;
                }
            }
            if (allMatch) return rule;
        }
        return null;
    }

    /**
     * 基于接口响应定义自动生成 Mock 数据
     */
    private String generateAutoMock(Long endpointId) {
        List<ApiResponse> responses = responseMapper.selectList(
                new LambdaQueryWrapper<ApiResponse>()
                        .eq(ApiResponse::getEndpointId, endpointId)
                        .eq(ApiResponse::getParentId, 0L)
                        .orderByAsc(ApiResponse::getSortOrder));

        if (responses.isEmpty()) {
            return "{\"code\": 0, \"msg\": \"success\", \"data\": null}";
        }

        Map<String, Object> result = new LinkedHashMap<>();
        for (ApiResponse resp : responses) {
            result.put(resp.getName(), generateMockValue(resp.getName(), resp.getType(), endpointId, resp.getId()));
        }

        try {
            return objectMapper.writerWithDefaultPrettyPrinter().writeValueAsString(result);
        } catch (Exception e) {
            return "{}";
        }
    }

    /**
     * 根据字段名和类型智能生成 Mock 值
     */
    private Object generateMockValue(String fieldName, String type, Long endpointId, Long parentId) {
        if (type == null) type = "string";
        ThreadLocalRandom random = ThreadLocalRandom.current();

        return switch (type.toLowerCase()) {
            case "integer", "int", "long" -> {
                if (fieldName.contains("id")) yield random.nextLong(1, 10000);
                if (fieldName.contains("age")) yield random.nextInt(18, 65);
                if (fieldName.contains("code")) yield 0;
                if (fieldName.contains("count") || fieldName.contains("total")) yield random.nextInt(0, 1000);
                yield random.nextInt(0, 100);
            }
            case "boolean", "bool" -> random.nextBoolean();
            case "number", "double", "float", "decimal" -> Math.round(random.nextDouble(0, 10000) * 100.0) / 100.0;
            case "string" -> {
                String lower = fieldName.toLowerCase();
                if (lower.contains("name")) yield "张三" + random.nextInt(100);
                if (lower.contains("email")) yield "user" + random.nextInt(1000) + "@example.com";
                if (lower.contains("phone") || lower.contains("mobile"))
                    yield "138" + String.format("%08d", random.nextInt(100000000));
                if (lower.contains("url") || lower.contains("link"))
                    yield "https://example.com/" + UUID.randomUUID().toString().substring(0, 8);
                if (lower.contains("time") || lower.contains("date"))
                    yield "2024-01-" + String.format("%02d", random.nextInt(1, 29)) + " 12:00:00";
                if (lower.contains("desc") || lower.contains("remark")) yield "这是一段自动生成的Mock描述文本";
                if (lower.contains("token")) yield UUID.randomUUID().toString();
                if (lower.equals("msg") || lower.equals("message")) yield "success";
                yield fieldName + "_" + random.nextInt(1000);
            }
            case "array", "list" -> {
                // 生成3个元素的数组
                List<Object> list = new ArrayList<>();
                // 查找子字段
                List<ApiResponse> children = responseMapper.selectList(
                        new LambdaQueryWrapper<ApiResponse>()

                                .eq(ApiResponse::getParentId, parentId)
```

```java
                new LambdaQueryWrapper<Module>()
                        .eq(Module::getPushToken, payload.getToken()));

            return PushResult.fail("无效的推送Token");

                        new LambdaQueryWrapper<ApiGroup>()

                                .eq(ApiGroup::getName, pushGroup.getName()));

                    group.setModuleId(module.getId());
                    group.setName(pushGroup.getName());
                    group.setDescription(pushGroup.getDescription());
                    group.setAuthor(pushGroup.getAuthor());
                    groupMapper.insert(group);
                } else {
                    group.setDescription(pushGroup.getDescription());
                    group.setAuthor(pushGroup.getAuthor());
                    groupMapper.updateById(group);
                }
                groupCount++;

                // 3. 遍历接口
                for (PushApiEndpoint pushEndpoint : pushGroup.getEndpoints()) {
                    // 根据 method + url 匹配现有接口
                    ApiEndpoint endpoint = endpointMapper.selectOne(
                            new LambdaQueryWrapper<ApiEndpoint>()
                                    .eq(ApiEndpoint::getModuleId, module.getId())
                                    .eq(ApiEndpoint::getHttpMethod, pushEndpoint.getHttpMethod())
                                    .eq(ApiEndpoint::getUrl, pushEndpoint.getUrl()));

                    if (endpoint == null) {
                        endpoint = new ApiEndpoint();
                        endpoint.setModuleId(module.getId());
                        endpoint.setGroupId(group.getId());
                    }

                    endpoint.setName(pushEndpoint.getName());
                    endpoint.setDescription(pushEndpoint.getDescription());
                    endpoint.setHttpMethod(pushEndpoint.getHttpMethod());
                    endpoint.setUrl(pushEndpoint.getUrl());
                    endpoint.setContentType(pushEndpoint.getContentType());
                    endpoint.setAuthor(pushEndpoint.getAuthor());
                    endpoint.setDeprecated(pushEndpoint.getDeprecated() ? 1 : 0);

                    if (endpoint.getId() == null) {
                        endpointMapper.insert(endpoint);
                    } else {
                        endpointMapper.updateById(endpoint);
                    }

                    // 4. 更新参数 (先删后插)
                    parameterMapper.delete(new LambdaQueryWrapper<ApiParameter>()
                            .eq(ApiParameter::getEndpointId, endpoint.getId()));
                    savePushParams(pushEndpoint.getParameters(), endpoint.getId(), 0L);

                    // 5. 更新响应
                    responseMapper.delete(new LambdaQueryWrapper<ApiResponse>()
                            .eq(ApiResponse::getEndpointId, endpoint.getId()));
                    savePushResponses(pushEndpoint.getResponses(), endpoint.getId(), 0L);

                    endpointCount++;
                }
            }

            return PushResult.success(groupCount, endpointCount);
        } catch (Exception e) {
            log.error("推送接收异常", e);
            return PushResult.fail("推送处理异常: " + e.getMessage());
        }
    }

    private void savePushParams(List<PushApiParam> params, Long endpointId, Long parentId) {
        if (params == null) return;
        for (int i = 0; i < params.size(); i++) {
            PushApiParam p = params.get(i);

            param.setEndpointId(endpointId);
            param.setParentId(parentId);
            param.setParamIn(p.getParamIn());
            param.setName(p.getName());
            param.setType(p.getType());
            param.setRequired(p.getRequired() ? 1 : 0);
            param.setDescription(p.getDescription());
            param.setExample(p.getExample());
            param.setDefaultValue(p.getDefaultValue());
            param.setMaxLength(p.getMaxLength());
            param.setSortOrder(i);
            parameterMapper.insert(param);

            if (p.getChildren() != null) {
                savePushParams(p.getChildren(), endpointId, param.getId());
            }
        }
    }

    private void savePushResponses(List<PushApiResponse> responses, Long endpointId, Long parentId) {
        if (responses == null) return;
        for (int i = 0; i < responses.size(); i++) {
            PushApiResponse r = responses.get(i);
            ApiResponse resp = new ApiResponse();
            resp.setEndpointId(endpointId);
            resp.setParentId(parentId);
            resp.setHttpCode(r.getHttpCode() != null ? r.getHttpCode() : 200);
            resp.setName(r.getName());
            resp.setType(r.getType());
            resp.setDescription(r.getDescription());
            resp.setExample(r.getExample());
            resp.setSortOrder(i);
            responseMapper.insert(resp);

            if (r.getChildren() != null) {
                savePushResponses(r.getChildren(), endpointId, resp.getId());
            }
        }
    }
```

```java
    private final ObjectMapper objectMapper;

    /**
     * 对比两个版本
     */
    public DiffResult compare(Long endpointId, String versionA, String versionB) throws Exception {
        ApiVersion snapshotA = versionMapper.findByEndpointAndVersion(endpointId, versionA);
        ApiVersion snapshotB = versionMapper.findByEndpointAndVersion(endpointId, versionB);

        if (snapshotA == null || snapshotB == null) {
            throw new IllegalArgumentException("版本快照不存在");
        }

        EndpointDetailVO a = objectMapper.readValue(snapshotA.getSnapshotData(), EndpointDetailVO.class);

        DiffResult result = new DiffResult();

        result.setVersionB(versionB);

        // 基本信息对比
        if (!Objects.equals(a.getUrl(), b.getUrl())) {
            result.addChange(new DiffItem("url", a.getUrl(), b.getUrl(), "breaking"));
        }
        if (!Objects.equals(a.getHttpMethod(), b.getHttpMethod())) {
            result.addChange(new DiffItem("httpMethod", a.getHttpMethod(), b.getHttpMethod(), "breaking"));
        }
        if (!Objects.equals(a.getName(), b.getName())) {
            result.addChange(new DiffItem("name", a.getName(), b.getName(), "non-breaking"));
        }
        if (!Objects.equals(a.getContentType(), b.getContentType())) {
            result.addChange(new DiffItem("contentType", a.getContentType(), b.getContentType(), "breaking"));
        }

        // 参数对比
        diffParams(result, "header", a.getHeaderParams(), b.getHeaderParams());
        diffParams(result, "path", a.getPathParams(), b.getPathParams());
        diffParams(result, "query", a.getQueryParams(), b.getQueryParams());
        diffParams(result, "body", a.getBodyParams(), b.getBodyParams());

        // 响应对比
        diffResponses(result, a.getResponses(), b.getResponses());

        return result;
    }

    private void diffParams(DiffResult result, String category,
                            List<ParamTreeVO> listA, List<ParamTreeVO> listB) {
        if (listA == null) listA = List.of();
        if (listB == null) listB = List.of();

        Map<String, ParamTreeVO> mapA = new LinkedHashMap<>();
        listA.forEach(p -> mapA.put(p.getName(), p));
        Map<String, ParamTreeVO> mapB = new LinkedHashMap<>();
        listB.forEach(p -> mapB.put(p.getName(), p));

        // 新增的参数
        for (String name : mapB.keySet()) {
            if (!mapA.containsKey(name)) {
                result.addChange(new DiffItem(category + ".param." + name, null,
                        mapB.get(name).getType(), "added"));
            }
        }

        // 删除的参数
        for (String name : mapA.keySet()) {
            if (!mapB.containsKey(name)) {
                result.addChange(new DiffItem(category + ".param." + name,
                        mapA.get(name).getType(), null, "removed"));
            }
        }

        // 变更的参数
        for (String name : mapA.keySet()) {
            if (mapB.containsKey(name)) {
                ParamTreeVO pa = mapA.get(name);
                ParamTreeVO pb = mapB.get(name);
                if (!Objects.equals(pa.getType(), pb.getType())) {
                    result.addChange(new DiffItem(category + ".param." + name + ".type",
                            pa.getType(), pb.getType(), "breaking"));
                }
                if (!Objects.equals(pa.getRequired(), pb.getRequired())) {
                    String severity = pb.getRequired() ? "breaking" : "non-breaking";
                    result.addChange(new DiffItem(category + ".param." + name + ".required",
                            String.valueOf(pa.getRequired()), String.valueOf(pb.getRequired()), severity));
                }
            }
        }
    }

    private void diffResponses(DiffResult result, List<ResponseTreeVO> listA, List<ResponseTreeVO> listB) {
        if (listA == null) listA = List.of();
        if (listB == null) listB = List.of();

        Map<String, ResponseTreeVO> mapA = new LinkedHashMap<>();
        listA.forEach(r -> mapA.put(r.getName(), r));
        Map<String, ResponseTreeVO> mapB = new LinkedHashMap<>();
        listB.forEach(r -> mapB.put(r.getName(), r));

        for (String name : mapB.keySet()) {
            if (!mapA.containsKey(name)) {
                result.addChange(new DiffItem("response." + name, null,
                        mapB.get(name).getType(), "added"));
            }
        }
        for (String name : mapA.keySet()) {
            if (!mapB.containsKey(name)) {
                result.addChange(new DiffItem("response." + name,
                        mapA.get(name).getType(), null, "removed"));
            }
        }
    }
}

@Data
public class DiffResult {
    private String versionA;
    private String versionB;
    private List<DiffItem> changes = new ArrayList<>();
    private int breakingCount;
    private int nonBreakingCount;
    private int addedCount;
    private int removedCount;

    public void addChange(DiffItem item) {
        changes.add(item);
        switch (item.getSeverity()) {
            case "breaking" -> breakingCount++;
            case "non-breaking" -> nonBreakingCount++;
            case "added" -> addedCount++;
            case "removed" -> removedCount++;
        }
    }
}

@Data
public class DiffItem {
    private String path;
    private String oldValue;
    private String newValue;
    private String severity; // breaking, non-breaking, added, removed

    public DiffItem(String path, String oldValue, String newValue, String severity) {
        this.path = path;
        this.oldValue = oldValue;
        this.newValue = newValue;
        this.severity = severity;
    }
}
```

```java
package com.apihub.config;

import io.github.bucket4j.Bandwidth;
import io.github.bucket4j.Bucket;
import io.github.bucket4j.Refill;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;

import java.time.Duration;
import java.util.Map;
import java.util.concurrent.ConcurrentHashMap;

@Configuration
public class RateLimitConfig {

    // 每用户每分钟最多30次调试请求
    private static final int DEBUG_RATE_LIMIT = 30;

    private final Map<Long, Bucket> userBuckets = new ConcurrentHashMap<>();

    public Bucket resolveBucket(Long userId) {
        return userBuckets.computeIfAbsent(userId, this::newBucket);
    }

    private Bucket newBucket(Long userId) {
        Bandwidth limit = Bandwidth.classic(DEBUG_RATE_LIMIT,
                Refill.intervally(DEBUG_RATE_LIMIT, Duration.ofMinutes(1)));
        return Bucket.builder()
                .addLimit(limit)
                .build();
```

```java
// ============================================================
// com/apihub/debug/DebugProxyService.java (增强版)
// ============================================================
package com.apihub.debug;

import com.apihub.common.context.UserContext;
import com.apihub.common.exception.BizException;
import com.apihub.config.RateLimitConfig;
import com.apihub.mapper.*;
import com.apihub.model.dto.debug.DebugRequestDTO;
import com.apihub.model.entity.*;
import com.apihub.model.vo.DebugResponseVO;
import com.fasterxml.jackson.databind.ObjectMapper;
import io.github.bucket4j.Bucket;
import io.github.bucket4j.ConsumptionProbe;
import lombok.RequiredArgsConstructor;
import lombok.extern.slf4j.Slf4j;
import org.springframework.beans.factory.annotation.Value;
import org.springframework.stereotype.Service;

import java.net.URI;
import java.net.http.*;
import java.time.Duration;
import java.util.*;
import java.util.regex.Matcher;
import java.util.regex.Pattern;

@Slf4j
@Service
@RequiredArgsConstructor
public class DebugProxyService {

    private final EnvironmentMapper environmentMapper;
    private final DebugHistoryMapper debugHistoryMapper;
    private final RateLimitConfig rateLimitConfig;
    private final ObjectMapper objectMapper;

    @Value("${apihub.debug.timeout:30}")
    private int defaultTimeoutSeconds;

    @Value("${apihub.debug.max-response-size:10485760}") // 10MB
    private long maxResponseSize;

    private final HttpClient httpClient = HttpClient.newBuilder()
            .connectTimeout(Duration.ofSeconds(10))
            .followRedirects(HttpClient.Redirect.NORMAL)
            .build();

    // 敏感字段脱敏正则
    private static final Pattern SENSITIVE_PATTERN = Pattern.compile(
            "(\"(?:password|pwd|secret|token|apiKey|api_key|accessToken)\"\\s*:\\s*\")[^\"]*(\")",
            Pattern.CASE_INSENSITIVE);

    public DebugResponseVO send(DebugRequestDTO request) {
        Long userId = UserContext.getUserId();

        // 1. 限流检查
        Bucket bucket = rateLimitConfig.resolveBucket(userId);
        ConsumptionProbe probe = bucket.tryConsumeAndReturnRemaining(1);
        if (!probe.isConsumed()) {
            long waitSeconds = probe.getNanosToWaitForRefill() / 1_000_000_000;
            throw new BizException(429, "请求过于频繁，请 " + waitSeconds + " 秒后重试");
        }

        // 2. 获取环境配置
        Environment env = null;
        if (request.getEnvId() != null) {
            env = environmentMapper.selectById(request.getEnvId());
        }

        // 3. 构建完整URL
        String fullUrl = buildFullUrl(request, env);

        // 4. 合并请求头
        Map<String, String> headers = mergeHeaders(request, env);

        // 5. 变量替换
        String body = resolveVariables(request.getBody(), env);
        headers.replaceAll((k, v) -> resolveVariables(v, env));

        // 6. 发送请求
        long startTime = System.nanoTime();
        DebugResponseVO response;

        try {
            HttpRequest httpRequest = buildHttpRequest(fullUrl, request.getMethod(), headers, body);
            HttpResponse<String> httpResponse = httpClient.send(httpRequest,
                    HttpResponse.BodyHandlers.ofString());

            long durationMs = (System.nanoTime() - startTime) / 1_000_000;

            // 7. 响应大小检查
            if (httpResponse.body() != null && httpResponse.body().length() > maxResponseSize) {
                response = DebugResponseVO.builder()
                        .statusCode(413)
                        .body("{\"error\": \"响应体过大，超过 " + (maxResponseSize / 1024 / 1024) + "MB 限制\"}")
                        .durationMs(durationMs)
                        .build();
            } else {
                response = DebugResponseVO.builder()
                        .statusCode(httpResponse.statusCode())
                        .headers(convertHeaders(httpResponse.headers().map()))
                        .body(httpResponse.body())
                        .durationMs(durationMs)
                        .bodySize(httpResponse.body() != null ? httpResponse.body().length() : 0)
                        .build();
            }
        } catch (HttpTimeoutException e) {
            long durationMs = (System.nanoTime() - startTime) / 1_000_000;
            response = DebugResponseVO.builder()
                    .statusCode(504)
                    .body("{\"error\": \"请求超时，超过 " + defaultTimeoutSeconds + " 秒\"}")
                    .durationMs(durationMs)
                    .build();
        } catch (Exception e) {
            log.error("Debug proxy error", e);
            long durationMs = (System.nanoTime() - startTime) / 1_000_000;
            response = DebugResponseVO.builder()
                    .statusCode(502)
                    .body("{\"error\": \"" + e.getMessage() + "\"}")
                    .durationMs(durationMs)
                    .build();
        }

        // 8. 异步保存历史 (脱敏处理)
        saveHistoryAsync(request, response, userId);

        return response;
    }

    private String buildFullUrl(DebugRequestDTO request, Environment env) {
        String baseUrl = "";
        if (env != null && env.getBaseUrl() != null) {
            baseUrl = env.getBaseUrl();
        }
        return baseUrl + request.getUrl();
    }

    private Map<String, String> mergeHeaders(DebugRequestDTO request, Environment env) {
        Map<String, String> headers = new LinkedHashMap<>();
        if (env != null && env.getGlobalHeaders() != null) {
            headers.putAll(env.getGlobalHeaders());
        }
        if (request.getHeaders() != null) {
            headers.putAll(request.getHeaders());
        }
        return headers;
    }

    private String resolveVariables(String text, Environment env) {
        if (text == null || env == null || env.getVariables() == null) {
            return text;
        }
        String result = text;
        for (Map.Entry<String, Object> entry : env.getVariables().entrySet()) {
            String placeholder = "{{" + entry.getKey() + "}}";
            result = result.replace(placeholder, String.valueOf(entry.getValue()));
        }
        return result;
    }

    private HttpRequest buildHttpRequest(String url, String method,
                                          Map<String, String> headers, String body) {
        HttpRequest.Builder builder = HttpRequest.newBuilder()
                .uri(URI.create(url))
                .timeout(Duration.ofSeconds(defaultTimeoutSeconds));

        headers.forEach(builder::header);

        if (body != null && !body.isEmpty() && List.of("POST", "PUT", "PATCH").contains(method)) {
            builder.method(method, HttpRequest.BodyPublishers.ofString(body));
        } else {
            builder.method(method, HttpRequest.BodyPublishers.noBody());
        }

        return builder.build();
    }

    private Map<String, String> convertHeaders(Map<String, List<String>> headers) {
        Map<String, String> result = new LinkedHashMap<>();
        headers.forEach((k, v) -> result.put(k, String.join("; ", v)));
        return result;
    }

    private String maskSensitiveData(String text) {
        if (text == null) return null;
        Matcher matcher = SENSITIVE_PATTERN.matcher(text);
        return matcher.replaceAll("$1***$2");
    }

    @org.springframework.scheduling.annotation.Async
    protected void saveHistoryAsync(DebugRequestDTO request, DebugResponseVO response, Long userId) {
        try {
            DebugHistory history = new DebugHistory();
            history.setProjectId(request.getProjectId());
            history.setEndpointId(request.getEndpointId());
            history.setUserId(userId);
            history.setMethod(request.getMethod());
            history.setUrl(request.getUrl());
            history.setRequestHeader(objectMapper.writeValueAsString(request.getHeaders()));
            // 请求体脱敏
            history.setRequestBody(maskSensitiveData(request.getBody()));
            history.setResponseCode(response.getStatusCode());
            history.setResponseHeader(objectMapper.writeValueAsString(response.getHeaders()));
            // 响应体脱敏
            history.setResponseBody(maskSensitiveData(response.getBody()));
            history.setDurationMs(response.getDurationMs());
            debugHistoryMapper.insert(history);
        } catch (Exception e) {
            log.warn("Failed to save debug history", e);
        }
    }
}
```

## 九、关键设计决策总结

| 决策点        | 选择                      | 理由                         |
|------------|-------------------------|----------------------------|
| **缓存**     | Caffeine优先, Redis可选     | 减少必要依赖, 单机部署够用             |
| **搜索**     | MySQL FULLTEXT (ngram)  | 不引入ES, 中小规模完全够用            |
| **文件存储**   | 本地磁盘                    | 简单可靠, 后续可接S3/MinIO         |
| **异步任务**   | Spring @Async           | 不引入MQ, 审计日志/历史保存用          |
| **版本快照**   | JSON列存储                 | MySQL 8.0 JSON支持好, 避免复杂关联  |
| **Mock引擎** | 内嵌在主服务                  | 不独立部署, 通过路径前缀区分            |
| **前端UI**   | Ant Design 5 + Tailwind | 组件完整 + 自定义样式灵活             |
| **代码编辑器**  | Monaco Editor           | VS Code同款, JSON编辑体验好       |
| **权限模型**   | 向上继承                    | Space→Project 权限继承, 减少管理成本 |
| **数据库迁移**  | Flyway                  | 版本化SQL管理, 部署时自动执行          |
